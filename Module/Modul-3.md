| :--- |
| **Stundenziel / "Warum?"** |

| Stundenplanung 17/8: Modul 3 Arbeitssession "Infografik & Data Storytelling" |

| **Stundenziel / "Warum?"** |

| **Kompetenzen** (gemäß Lehrplan AHS Informatik, 5. Klasse) |

Der Text soll Neugier wecken.'
